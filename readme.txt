Disable Shipments when the free one is active


== Description ==

The plugin deactivates the sending option when free shipping is activated.

<strong>No options</strong>, just activate the plugin and it’s done.

== Installation ==

1. Go to your WP Dashboard > Plugins and search for ‘Disable Shipments when the free one is active’ or…
2. Download the plugin from WP repository.
3. Upload the plugin folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
